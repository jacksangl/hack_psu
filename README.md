# GeoScope

## Frontend

Built with React, TypeScript, Vite, Tailwind CSS, and Three.js.

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- npm

### Setup

```bash
cd frontend/geoscope
npm install
```

### Development

```bash
npm run dev
```

Starts the dev server at `http://localhost:5173`.

### Build

```bash
npm run build
```

Output is in `frontend/geoscope/dist/`.

### Preview Production Build

```bash
npm run preview
```
